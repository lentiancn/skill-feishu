# 部门访问权限配置

## 部门可见范围

在飞书开放平台中配置「部门可见范围」来控制应用可以访问哪些部门的数据。

## 配置步骤

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 进入你的应用
3. 在左侧菜单找到「应用权限」→「通讯录权限」→「部门可见范围」
4. 点击「添加部门」
5. 选择要授权的部门

## 部门 ID 说明

| 部门 | 部门 ID | 说明 |
|------|---------|------|
| 根部门 | `0` | 所有部门的父部门 |
| 指定部门 | `od_xxx` | 具体的部门 ID |

## 权限校验逻辑

| 请求方式 | 权限校验 |
|---------|---------|
| 携带 `department_id` | 检查应用是否有该部门的访问权限 |
| 不携带 `department_id` | 返回权限范围内的独立用户 |

## 注意事项

⚠️ 如果应用没有部门权限，调用用户列表接口时会返回 `40004` (no dept authority error) 错误。
