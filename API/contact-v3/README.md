# contact-v3 API

通讯录 v3 API 用于操作用户和部门信息。

## API 列表

| API | 说明 | 状态 |
|-----|------|------|
| `users.md` | 获取用户列表（历史版本） | ✅ 已记录 |
| `find_by_department.md` | 获取部门直属用户列表（推荐） | ✅ 已记录 |

## 文档地址

- [通讯录 v3 API 文档](https://open.feishu.cn/document/server-docs/contact-v3)

## 注意事项

⚠️ `contact/v3/users` 接口已为历史版本，不再维护更新，不推荐使用。推荐使用 `contact/v3/users/find_by_department` 接口。
