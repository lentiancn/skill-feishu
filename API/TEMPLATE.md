# API 模板文档

## 说明

这是 API 文档模板。使用此模板创建新的 API 文档。

## 结构

- **注意事项**
- **权限要求**
- **接口信息**
- **接口请求**
  - 请求头
  - 请求参数
  - 请求体
- **接口响应**
  - 响应头
  - 响应体
  - 响应数据结构
- **请求示例**
- **响应示例**
- **错误码**
- **相关链接**

## 注意事项

- 所有文档必须保持飞书官方文档的完整性，禁止简化、省略、合并
- 所有示例必须完整展示，禁止省略部分行
- 所有字段必须完整说明权限要求，禁止推算或猜测
- 所有内容必须真实记录，禁止偷懒、失误或简化
